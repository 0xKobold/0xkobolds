## 📱 Multi-Channel Support

### WhatsApp (Baileys)

```typescript
import { getWhatsAppIntegration } from "0xkobold/channels";

const whatsapp = getWhatsAppIntegration({
  sessionPath: "~/.0xkobold/whatsapp-session",
});

// Start and scan QR code
await whatsapp.start();

// Send message
await whatsapp.sendText("1234567890", "Hello from 0xKobold!");

// Send to group
await whatsapp.sendText("group-id@g.us", "Group message");
```

**CLI:**
```bash
0xkobold whatsapp start  # Show QR code
0xkobold whatsapp status
0xkobold whatsapp send <number> <message>
```

**Features:**
- QR code pairing
- Individual messages
- Group support
- Auto-reconnect
- Media handling (images, audio)

### Telegram

```typescript
import { getTelegramIntegration } from "0xkobold/channels";

const telegram = getTelegramIntegration({
  token: process.env.TELEGRAM_BOT_TOKEN,
  mode: "polling", // or "webhook"
});

await telegram.start();

// Send with formatting
await telegram.sendMessage(chatId, "Hello!", {
  parseMode: "Markdown",
  buttons: [[{ text: "Button", callbackData: "click" }]],
});
```

**CLI:**
```bash
TELEGRAM_BOT_TOKEN=xxx 0xkobold telegram start
0xkobold telegram send <chatId> <message>
```

## 🛡️ Security

### Device Authentication

```typescript
import { getDeviceAuth } from "0xkobold/auth";

const auth = getDeviceAuth();

// Initialize device
const device = await auth.initialize("My Laptop");

// Generate token
const token = auth.generateToken(["read", "write"]);

// Validate
const { valid } = auth.validateToken(token.token);
```

**Features:**
- Multi-device support
- Token-based auth
- Token expiration
- Device revocation

---

### Docker Sandbox

```typescript
import { getDockerRunner } from "0xkobold/sandbox";

const runner = getDockerRunner({
  image: "node:20-slim",
  memoryLimit: "512m",
  cpuLimit: "1.0",
  network: "none",  // Isolated
});

// Run in container
const result = await runner.run({
  command: "node",
  args: ["-e", "console.log('Hello')"],
  timeout: 30000,
});

console.log(result.stdout); // Hello
```

**Security Features:**
- Network isolation (none by default)
- Memory limits
- CPU limits
- Volume mounting
- Auto-cleanup

## 🖼️ Media Support

### Vision (Image Analysis)

```typescript
import { VisionAnalyzer } from "0xkobold/media";

const analyzer = new VisionAnalyzer({
  provider: "claude",
  apiKey: process.env.ANTHROPIC_API_KEY,
});

const result = await analyzer.analyzeImage("./image.png");

console.log(result.description); // Image description
console.log(result.objects);      // Detected objects
```

**Supported:**
- Claude Vision
- Base64 encoding
- URL downloads
- Object detection

---

### Audio Transcription

```typescript
import { AudioTranscriber } from "0xkobold/media";

const transcriber = new AudioTranscriber({
  provider: "openai",
  apiKey: process.env.OPENAI_API_KEY,
});

const result = await transcriber.transcribeFile("./audio.mp3");

console.log(result.text);
console.log(result.segments); // Timestamps
```

**Features:**
- Whisper integration
- Voice notes
- Timestamped segments
- Format conversion
